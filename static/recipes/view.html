#3: Cocoon 'em in gaffer tape

Description

i am a big believer of starting with the html/css for features before plugging in the functionality. that way, you can agree to the interface before worrying about functionality.

include:

  • basic app layout
  • multiple boards with add/remove/rename
  • multiple columns for lists with add/remove/rename
  • collapsed items in lists
  • item detail modal

more as we flesh out the first milestone features
